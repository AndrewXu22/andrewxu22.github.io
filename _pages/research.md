## Research Overview

My research focuses on designing and building systems that support AI applications,
distributed machine learning, human-robot collaboration, and large-scale data processing.

---

## Current Project

### SpamX
Next-generation system for efficient scalable social spam detection on large-scale online social data streams.

---

## Past Projects

### SR3 – Stateful Stream Recovery

SR3 is a customizable recovery framework for distributed stream processing systems.
It provides scalable mechanisms for protecting large distributed states and
supports star, line, and tree structured recovery architectures.

---

### SpamHunter

SpamHunter is a scalable system for detecting social spam across distributed data sources
by exploiting correlations among spam behaviors.

---

### GPU Unified Memory Optimization

A machine-learning guided system that optimizes when to use unified memory
versus discrete GPU memory for CUDA applications.
